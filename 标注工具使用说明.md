# 数据集标注工具使用说明

**作者：同济子豪兄**  
**项目地址：https://github.com/TommyZihao/zihao_commercial**

## 🎯 功能特点

- **多任务支持**：支持图像分类、目标检测、语义分割三种标注任务
- **现代化界面**：基于Bootstrap 5的美观响应式界面
- **多格式导出**：支持COCO、YOLO等主流数据集格式
- **实时预览**：Canvas绘图，支持缩放、平移等操作
- **项目管理**：完整的项目创建、管理、删除功能
- **快捷键支持**：提高标注效率的快捷键操作

## 📋 系统要求

- Python 3.7+
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 至少2GB内存

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
python annotation_tool.py
```

### 3. 访问应用

打开浏览器访问：http://localhost:5000

## 📖 详细使用教程

### 创建项目

1. 在首页点击"创建新项目"按钮
2. 填写项目名称
3. 选择标注任务类型：
   - **图像分类**：为整张图像标注类别
   - **目标检测**：绘制边界框标注目标位置
   - **语义分割**：像素级标注（开发中）
4. 输入类别列表，每行一个类别名称
5. 点击"创建项目"

### 上传图像

1. 进入项目页面
2. 点击"上传图像"按钮
3. 选择或拖拽图像文件
4. 支持格式：JPG, PNG, GIF, BMP, TIFF, WebP
5. 点击"上传"

### 图像分类标注

1. 点击图像进入标注页面
2. 在右侧工具面板选择对应的类别
3. 按Ctrl+S保存标注

### 目标检测标注

1. 点击图像进入标注页面
2. 在"当前类别"下拉框中选择要标注的类别
3. 在图像上按住鼠标左键拖拽绘制边界框
4. 在右侧对象列表中可以查看、选择、删除标注对象
5. 按Ctrl+S保存标注

### 导出数据

1. 在项目页面点击"导出"下拉菜单
2. 选择导出格式：
   - **COCO格式**：适用于目标检测任务
   - **YOLO格式**：适用于YOLO系列模型
   - **分类格式**：适用于图像分类任务
3. 文件将自动下载

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存标注 |
| Ctrl+Z | 撤销上一步操作 |
| Delete | 删除选中的标注对象 |
| 鼠标滚轮 | 缩放图像 |

## 🔧 配置说明

### 修改端口

在`annotation_tool.py`文件末尾修改：

```python
app.run(debug=True, host='0.0.0.0', port=5000)  # 修改port参数
```

### 修改上传限制

在`annotation_tool.py`中修改：

```python
app.config['MAX_CONTENT_LENGTH'] = 16 * 1024 * 1024  # 16MB
```

### 添加支持的图像格式

在`annotation_tool.py`中修改：

```python
ALLOWED_EXTENSIONS = {'png', 'jpg', 'jpeg', 'gif', 'bmp', 'tiff', 'webp'}
```

## 📊 数据格式说明

### COCO格式

导出的COCO格式包含以下字段：

```json
{
  "info": {...},
  "licenses": [...],
  "images": [
    {
      "id": 1,
      "width": 640,
      "height": 480,
      "file_name": "image.jpg"
    }
  ],
  "annotations": [
    {
      "id": 1,
      "image_id": 1,
      "category_id": 1,
      "bbox": [x, y, width, height],
      "area": 12800,
      "iscrowd": 0
    }
  ],
  "categories": [
    {
      "id": 1,
      "name": "cat",
      "supercategory": ""
    }
  ]
}
```

### YOLO格式

每张图像对应一个txt文件，格式为：

```
class_id x_center y_center width height
```

坐标为归一化值（0-1之间）。

### 分类格式

简单的JSON格式：

```json
{
  "image1.jpg": "cat",
  "image2.jpg": "dog"
}
```

## 🐛 常见问题

### Q: 上传图像失败
A: 检查图像格式是否支持，文件大小是否超过16MB限制

### Q: 标注保存失败
A: 确保有写入权限，检查磁盘空间是否充足

### Q: 浏览器兼容性问题
A: 建议使用Chrome、Firefox等现代浏览器

### Q: 如何备份标注数据
A: 复制整个`annotations`文件夹即可
