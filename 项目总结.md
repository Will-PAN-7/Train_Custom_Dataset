# 数据集标注工具 - 项目总结

## 🎯 项目概述

成功创建了一个功能完整的数据集标注工具，支持图像分类、目标检测、语义分割三种计算机视觉任务的标注。

**作者：同济子豪兄**

## 📋 已完成功能

### ✅ 核心功能
- [x] **图像分类标注** - 为整张图像选择类别标签
- [x] **目标检测标注** - 绘制边界框标注目标位置
- [x] **语义分割标注** - 像素级标注（基础框架，可扩展）
- [x] **项目管理** - 创建、查看、删除项目
- [x] **图像上传** - 支持多种格式，拖拽上传
- [x] **数据导出** - COCO、YOLO、分类格式

### ✅ 用户界面
- [x] **现代化设计** - 基于Bootstrap 5的响应式界面
- [x] **直观操作** - Canvas绘图，实时预览
- [x] **快捷键支持** - 提高标注效率
- [x] **缩放功能** - 支持图像缩放、平移
- [x] **标注管理** - 对象列表、选择、删除

### ✅ 技术特性
- [x] **Web架构** - Flask后端 + HTML/CSS/JS前端
- [x] **数据持久化** - JSON格式存储标注数据
- [x] **多格式支持** - JPG, PNG, GIF, BMP, TIFF, WebP
- [x] **实时保存** - 标注数据实时保存到服务器

## 🔧 技术栈

### 后端技术
- **Flask 2.3.3** - Web框架
- **OpenCV 4.8.1** - 图像处理
- **Pillow 10.0.1** - 图像操作
- **NumPy 1.24.3** - 数值计算

### 前端技术
- **Bootstrap 5.1.3** - UI框架
- **Bootstrap Icons** - 图标库
- **HTML5 Canvas** - 图像绘制
- **JavaScript ES6** - 交互逻辑

## 🚀 核心功能详解

### 1. 项目管理系统
- 创建项目时可选择任务类型和类别
- 项目列表显示统计信息
- 支持项目删除和导出

### 2. 图像分类标注
- 简单的类别选择界面
- 一键保存标注结果
- 支持快捷键操作

### 3. 目标检测标注
- 鼠标拖拽绘制边界框
- 实时显示对象列表
- 支持对象选择、删除
- 边界框可视化标签

### 4. 数据导出系统
- **COCO格式**：包含完整的图像信息、标注信息、类别信息
- **YOLO格式**：归一化坐标，适配YOLO训练
- **分类格式**：简单的图像-类别映射
