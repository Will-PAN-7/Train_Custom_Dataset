# 📊 IMU数据标注功能使用指南

## 🎯 功能概述

本工具现已扩展支持IMU（惯性测量单元）数据的标注功能，可以对时序传感器数据进行动作分类和时间段标记。

## 🚀 快速开始

### 1. 切换到IMU模式
- 在标注工具界面中，点击右侧的"📊 IMU数据标注"按钮
- 界面将切换到IMU数据标注模式

### 2. 上传IMU数据文件
支持的文件格式：
- **CSV文件** (.csv): 标准的逗号分隔值文件
- **JSON文件** (.json): JSON格式的数据文件
- **TXT文件** (.txt): 空格或逗号分隔的文本文件

#### CSV文件格式示例：
```csv
timestamp,acc_x,acc_y,acc_z,gyro_x,gyro_y,gyro_z
0.00,0.12,-0.34,9.78,0.01,-0.02,0.03
0.01,0.15,-0.31,9.81,0.02,-0.01,0.02
...
```

#### JSON文件格式示例：
```json
[
  {"timestamp": 0.00, "acc_x": 0.12, "acc_y": -0.34, "acc_z": 9.78},
  {"timestamp": 0.01, "acc_x": 0.15, "acc_y": -0.31, "acc_z": 9.81},
  ...
]
```

### 3. 数据可视化
- 上传成功后，系统会自动生成时序图表
- 图表显示加速度计的X、Y、Z轴数据
- 不同轴用不同颜色表示：红色(X轴)、绿色(Y轴)、蓝色(Z轴)

### 4. 添加标注
有两种方式添加时间段标注：

#### 方式一：手动输入
1. 在"动作标签"输入框中输入动作名称（如：走路、跑步、静止等）
2. 在"开始时间"和"结束时间"输入框中输入时间范围（单位：秒）
3. 点击"添加时间段标注"按钮

#### 方式二：图表交互（未来版本）
- 在时序图表上直接拖拽选择时间段
- 系统会自动填充时间范围

## 📁 数据管理

### 保存标注
- 标注数据会自动保存到服务器
- 每个IMU文件的标注独立存储

### 导出标注数据
支持两种导出格式：

#### JSON格式
```json
{
  "export_time": "2024-01-01T12:00:00Z",
  "total_files": 1,
  "annotations": [
    {
      "file_id": "uuid-string",
      "annotations": [
        {
          "id": "imu_timestamp",
          "label": "walking",
          "start_time": 5.0,
          "end_time": 10.5,
          "timestamp": "2024-01-01T12:00:00Z"
        }
      ]
    }
  ]
}
```

#### CSV格式
```csv
file_id,start_time,end_time,label,description,timestamp
uuid-1,5.0,10.5,walking,,2024-01-01T12:00:00Z
uuid-1,15.2,20.8,running,,2024-01-01T12:05:00Z
```

## 🔧 API接口

### 上传IMU文件
```http
POST /api/imu/upload
Content-Type: multipart/form-data

file: [IMU数据文件]
```

### 获取IMU文件列表
```http
GET /api/imu/list
```

### 获取IMU数据
```http
GET /api/imu/{file_id}
```

### 获取/保存标注
```http
GET /api/imu/annotations/{file_id}
POST /api/imu/annotations/{file_id}
Content-Type: application/json

{
  "annotations": [...]
}
```

### 导出标注数据
```http
GET /api/imu/export/json
GET /api/imu/export/csv
```

## 💡 使用技巧

1. **时间精度**：建议使用小数点后1位的时间精度（如：5.2秒）
2. **标签命名**：使用有意义的标签名称，便于后续分析
3. **时间段重叠**：避免时间段重叠，确保标注的准确性
4. **数据质量**：上传前检查数据格式，确保时间戳连续

## 🔍 故障排除

### 常见问题

**Q: 上传文件失败？**
A: 检查文件格式是否支持，确保CSV文件包含数值数据

**Q: 图表显示异常？**
A: 检查数据中是否包含非数值内容，清理数据后重新上传

**Q: 标注保存失败？**
A: 确保时间范围有效（结束时间 > 开始时间）

**Q: 导出数据为空？**
A: 确认已添加标注数据，检查是否选择了正确的IMU文件
